# Kiyay Koki

## Clone and run
```bash
git clone https://github.com/kite1412/react-kiyay-koki.git
cd react-kiyay-koki
npm install
npm run dev
```

## Pull latest updates (only for cloned repo)
```bash
git pull origin main
```
